#  QR Code Generator using Python

This project is a simple **QR Code Generator** built with Python.  
It takes a **URL or text input from the user** and generates a **custom QR code image**.

---

##  Features
- Generates QR codes for URLs or text
- Auto-open QR after generation
- Timestamp-based filename (to avoid overwrite)
- Custom file name
- Custom colors
- Easy to use and beginner-friendly

---

##  Technologies Used
- Python 3
- qrcode library
- Pillow (for image generation)

---
